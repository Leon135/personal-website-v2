## About This Project

A beautiful weather dashboard that provides accurate forecasts and current conditions for any location.

### Key Features

- **Location Search** - Find weather for any city worldwide
- **7-Day Forecast** - Plan your week with extended forecasts
- **Interactive Charts** - Visualize temperature and precipitation trends
- **Geolocation** - Automatically detect user location

### API Integration

The app integrates with the OpenWeather API to fetch real-time weather data. Results are cached to minimize API calls and improve performance.
