## About This Project

This portfolio website showcases my work and skills as a developer. Built with modern web technologies, it features a clean, minimalist design with smooth animations and a dark theme.

### Key Features

- **Responsive Design** - Looks great on all devices
- **Dark Theme** - Easy on the eyes, especially for developers
- **Smooth Animations** - Subtle motion effects using Framer Motion
- **Markdown Support** - Projects are managed via markdown files

### Technical Details

The site is built with Next.js 15 and uses the App Router for optimal performance. Tailwind CSS provides the styling foundation, while Framer Motion handles animations.

The project system uses gray-matter to parse markdown frontmatter, making it easy to add new projects without touching the code.
